技术实现方案 配置BT面板及LNMP/LAMP环境  配置BT面板可参考官网宝塔面板， 在shell中执行如下命令即可在ubuntu中安装: wget -O install.sh http://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh 从安装返回警告信息可见宝塔面板默认的开发运行环境是python2的，但py将于2020年停止维护， 有些安装教程说可以设置py3为其默认环境，但这是作死节奏，切换为py3环境后或将出现各种异常而不能运行: 参考:升级云服务器的python3,导致宝塔面板打不开  安装成功后，按如下提示登录管理: 借助宝塔面板对服务器、数据库及网站进行管理，对于出现的各种错误，可以在/www/wwwlogs下找到相关日志信息 推荐选择LAMP环境，相关配置项考虑如下:  MySQL5.5 建议安装内存小于1GB的；MySQL 5.6 建议安装内存大于1GB；MySQL 5.7 建议安装内存大于2GB,由于测试服务器内存就是1G，为此我安装mysql5.5版本 注意nextcloud从13.0版本开始才支持php 7.2，由于我将下载的nextcloud server版本为16.0，因此我选择安装php 7.2  选择安装后会看到如下任务列表:
